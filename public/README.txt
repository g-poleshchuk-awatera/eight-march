This directory should contain the following image files:

1. flower1.png - Small decorative flower for the top-left corner of the form
2. flower2.png - Small decorative flower for the bottom-right corner of the form
3. flower3.png - Larger decorative flower for the top-left corner of the greeting card
4. flower4.png - Larger decorative flower for the bottom-right corner of the greeting card
5. floral-bg.png - Optional background pattern with floral elements

You can use free resources from sites like Freepik, Pixabay, or create custom ones.
The images should be in transparent PNG format and match the pink/feminine color scheme.
